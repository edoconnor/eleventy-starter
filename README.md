# Eleventy Starter

Minimal Eleventy starter template with `src/_includes` structure.

## Structure

```
src/
  _includes/
    partials/
      header.njk
      footer.njk
    base.njk
  assets/
    styles.css
  index.md
  about.md
.eleventy.js
package.json
```

## Usage

```bash
npm install
npm start     # dev server at localhost:8080
npm run build # output to /dist
```
